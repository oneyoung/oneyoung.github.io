导出骑记(iriding.cc)轨迹

导出骑记轨迹, 保存成.gpx格式. 方便各类GPS党制作路书, 或者导入到别的平台.

  • 依赖: gpxpy

    • https://github.com/tkrajina/gpxpy
    • 安装: pip2 install gpxpy

  • 用法:

    • ./iriding2gpx.py ROUTEID
    • ROUTEID 是每个轨迹的ID
      • 比如这个链接: http://iriding.cc/user/route/207130.shtml, route id 就是207130
  • 代码: iriding2gpx.py
